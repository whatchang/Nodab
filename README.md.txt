이 프로젝트는 Test와 test2가 있습니다.

Test - 파일을 기준디렉토리에서 최하위디렉토리까지 모든 
파일을 AES256으로 암호화합니다. 그리고 원본 파일을 삭제
해줍니다.


test2 - 사용자가 .enc(암호화된 파일)을 클릭시 복호화를 시켜주고
원본파일에 맞는 확장자프로그램을 실행시켜주어서 원본 프로그램을
실행시켜줍니다.
